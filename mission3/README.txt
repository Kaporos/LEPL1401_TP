# Mission 3

## Utilisation du programme

Il suffit d'executer main_standalone.py et une fenêtre s'ouvrira.

Vous pouvez cliquer n'importe où pour dessiner des drapeaux , changer le type de drapeau via le select, et changer la taille dans l'input a coté.

Afin de valider le changement de taille, il faut bien appuyer sur **enter** et si la valeur entrée est invalide vous serez prévenu.
